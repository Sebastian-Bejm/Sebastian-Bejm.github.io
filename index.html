About Me

I am Sebastian Bejm, a graduate from BCIT's Bachelor of Technology Computer Systems (Games Development option) program. I have mainly worked on projects using C#, C++ and Python, with mild exposure to web development languages. I am passionate about programming and aim to further improve my skills and learn new programming languages and related technologies for software development.

My Projects

Keyboard Warriors (Unity Game)

Collaborated with a team of four to create a turn-based RPG for Windows, developed with Unity engine. The player gets to fight through waves of enemies in battles, and they level up in the process.

Edgar the Exterminator (C++ Game)

Collaborated with a team to design and program an arcade platformer for Windows, developed with C++ and the OpenGL API. The player controls a character that can move and jump around, while shooting roaches that fall from the top of the screen, and collecting stars that appear which adds to the player's score.

Machine Learning implementation for Real-time Pathfinding

This project was an attempt to implement a machine learning algorithm using C++ as an application of real-time pathfinding in a simple game based on navigating randomly generated mazes with obstacles. The algorithm used was the Q-Learning algorithm, in which data is passed to the AI model to determine where to go in the maze.
